# 🛍️ Zaptro – Modern E-Commerce Web Application

Zaptro is a fully responsive E-Commerce web application built using React.js.  
It fetches product data from an external API and provides complete shopping functionality including filtering, category-wise browsing, dynamic routing, authentication, and cart management.

---
# View Live - https://yourzaptro.netlify.app/

## 🚀 Features

### 🛒 Product Management
- Fetches products dynamically from an external API
- Displays all available products
- Dynamic Single Product Page (`/product/:id`)
- Category-wise product listing
- Product filtering functionality
- Fully responsive design (Mobile, Tablet, Desktop)

### 🔐 Authentication
- Secure Login & Signup using Clerk
- Protected routes for authenticated users

### 🧠 State Management
- Context API for global state management
- Cart management using Context API
- Real-time cart updates

### 💾 Data Persistence
- LocalStorage integration
- Cart data persists after page refresh

### 🔄 Routing
- React Router DOM for navigation
- Dynamic routing for individual product pages
- Category-based routing
